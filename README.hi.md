# मन्नमित्र (MANNMITRA) — डेमो गाइड / README (हिंदी)

**मन्नमित्र** (मन मित्र — आपके मन का दोस्त) एक गोपनीय, स्तरबद्ध मानसिक स्वास्थ्य
प्लेटफ़ॉर्म है जो कॉलेज छात्रों के लिए बनाया गया है। पहला कदम भारी नहीं होना चाहिए —
इसीलिए यहाँ **कोई साइन-अप नहीं, कोई पहचान नहीं (No PII)**।

## इसमें क्या है
- **एआई ट्राइज** — मूड + 3 सवाल → गंभीरता (low/medium/high) और रूटिंग
- **पीयर सपोर्ट** — गुमनाम रूम + 1:1 श्रोता
- **काउंसलर + एडमिन** — शेयर की गई पूल, गुमनाम बुकिंग, प्रभाव रिपोर्ट
- **SOS / संकट** — दबाकर कॉल → Tele-MANAS 14416
- **डैशबोर्ड, जर्नल, CBT, ब्रीदिंग, कम्युनिटी, संसाधन**
- **भाषाएँ** — English · हिंदी · मराठी · ગુજરાતી
- **थीम** — डार्क मोड + हाई-कॉन्ट्रास्ट
- **PWA** — इनस्टॉल करें, ऑफ़लाइन चलाएँ (मोबाइल/टैब/लैपटॉप)

## कैसे चलाएँ
1. `index.html` खोलें (सबसे अच्छा: किसी लोकल सर्वर से, ताकि PWA/Service Worker काम करे)।
   ```
   python -m http.server 8080
   ```
   फिर ब्राउज़र में `http://localhost:8080` खोलें।
2. "Start anonymously" दबाएँ → ऑनबोर्डिंग टूर शुरू होगा।
3. `checkin.html` पर मूड + सवाल भरें → ट्राइज रिज़ल्ट देखें।

## प्राइवेसी
- कोई नाम / ईमेल / फ़ोन नहीं। सिर्फ़ अस्पष्ट `anon_id`।
- डेमो में डेटा आपके डिवाइस (localStorage) पर रहता है।
- "My Data" पेज से एक क्लिक में सब डिलीट करें।
